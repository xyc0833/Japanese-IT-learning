# -IT-
このリポジトリは、私の日本語学習の記録です。特に以下の3点を中心に、学んだ内容や気づきを整理しています。  ① 日本語タイピング：毎日のタイピング速度（WPM）や精度、よく間違える単語などをログとして記録。 ② ビジネス日本語：敬語・メール文例・上司や取引先との会話表現など、実務で使えるフレーズを中心にまとめています。 ③ IT業界の日本語：エンジニア現場で使われるカタカナ語・仕様書の表現・英語から日本語への翻訳メモなどを収録。  学習内容をただ保存するだけでなく、同じく日本語を学ぶエンジニアや社会人の方にも参考になるよう、できるだけ整理して残しています。毎日の小さな積み重ねが、確かな日本語力につながると信じて。
